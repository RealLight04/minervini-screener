# 📈 Minervini Stock Screener

Mark Minervini의 **SEPA(Trend Template + 실적 모멘텀 + VCP)** 방법론을 코드로 구현한 미국 주식 종목 발굴기.
매일 S&P 500 종목을 스크리닝해 **매수 후보**와 **매도 경고**를 신호로 제시합니다.

> ⚠️ 교육·참고용 도구입니다. 투자 권유가 아니며 모든 판단과 책임은 본인에게 있습니다.

## 스크리닝 로직

### 1단계 · 기술적 추세 (Stage 2 Uptrend) — 8개 조건
1. 현재가 > 150일 이동평균
2. 현재가 > 200일 이동평균
3. 150일 MA > 200일 MA
4. 200일 MA가 최소 1개월 상승 추세
5. 현재가 > 50일 MA
6. 현재가 ≥ 52주 저가 × 1.30 (저점 대비 +30% 이상)
7. 현재가 ≥ 52주 고가 × 0.75 (고점 대비 25% 이내)
8. RS(상대강도) 순위 상위 30% 이내

### 2단계 · 펀더멘털 (실적 모멘텀)
- 최근 분기 EPS 성장률 ≥ 25% (YoY)
- 최근 분기 매출 성장률 ≥ 20% (YoY)
- 연간 EPS 우상향 (최근 3년)

### 3단계 · VCP (변동성 축소 패턴)
- 조정폭이 좌→우로 줄어드는 수축 구조 탐지
- 마지막 수축 고점 = **피벗(매수 트리거 가격)**

### 매수/매도 신호
| 신호 | 의미 | 조건 |
|---|---|---|
| 적극 매수 | VCP 피벗 돌파 | 기술+펀더멘털+VCP 통과 & 현재가 ≥ 피벗 |
| 매수 | 매수 후보 | 기술+펀더멘털 통과 (피벗 돌파 대기) |
| 관심 | 추세는 양호, 실적 부족 | 기술 통과, 펀더멘털 미달 |
| 매도 | 추세 약화 조짐 | Stage 2 유지하나 50일선 하향 이탈 |
| 회피 | 약세 구간 | 200일선 하회 또는 MA 역배열 |

매수 신호에는 **피벗(매수가)** 와 **손절가(피벗 -8%, Minervini 기준)** 를 함께 제시합니다.

## 기술 스택
- **백엔드**: FastAPI + SQLAlchemy
- **데이터**: yfinance (주가·재무), Wikipedia (S&P 500 목록)
- **분석**: pandas / numpy
- **프론트**: Jinja2 (서버사이드 렌더링)
- **DB**: SQLite (스냅샷 동봉) / PostgreSQL 호환
- **배포**: Render

## 로컬 실행
```bash
pip install -r requirements.txt

# 데이터 수집 + 스크리닝 (S&P 500 전체, 약 15~20분)
python scripts/daily_update.py

# 웹서버 실행
./run.sh          # 또는: uvicorn main:app --host 0.0.0.0 --port 8001
```

## 배포 (Render)
- `render.yaml` 포함. 무료 플랜은 디스크가 임시라 **스케줄러를 끄고**(`ENABLE_SCHEDULER=false`),
  깃에 동봉한 `screener.db` 스냅샷을 그대로 서빙합니다.
- 데이터 갱신은 로컬에서 `daily_update.py` 실행 후 DB를 커밋하거나, GitHub Actions cron으로 자동화할 수 있습니다.
